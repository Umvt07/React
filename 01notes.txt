# 4. useState
Used to create state.

const [count, setCount] = useState(0)


- `count` → current value
- `setCount` → updates value
- `0` → initial value

# 5. State
State is special React data.
Changing state re-renders UI.
```js
setCount(count + 1)
```
Normal variable change does NOT update UI.

```js
count++
```
❌ Wrong
---

# 8. Dynamic Values in JSX

# 9. Re-rendering
When state changes:
```js

```

1. Updates state
